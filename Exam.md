## #1 
### В ящике 15 красных, 9 синих и 6 зеленых шаров. Наудачу вынимают 6 шаров. Какова вероятность того, что вынуты 1 зеленый, 2 синих и 3 красных шара?
Общее количество способов вытащить 6 шаров из ящика равно :point_down:
```
C(30, 6) = 30! / (6! * (30 - 6)!) = 593775
```
Количество способов вытащить 1 зеленый, 2 синих и 3 красных шара :point_down:
```
C(6, 1) * C(9, 2) * C(15, 3) =
  = (6! / (1! * (6 - 1)!)) * (9! / (2! * (9 - 2)!)) *
  * (15! / (3! * (15 - 3)!)) =
  = 6 * 36 * 455 = 98280
P = (количество способов вытащить 1 зеленый, 2 синих и 3 красных шара) /
  / (общее количество способов вытащить 6 шаров) =
  = 98280 / 593775 ≈
  ≈ 0.1655 = 16.55%
```

## #2
### Из букв слова РОТОР, составленного с помощью разрезной азбуки, наудачу последовательно извлекают 3 буквы и кладут в ряд. Какова вероятность того, что появится слово ТОР?
Так как у нас 6 букв в слове "РОТОР" и мы извлекаем 3 буквы :point_down:
```
C(6, 3) = 6! / (3! * (6-3)!) = 20
```
Вероятность появления слова "ТОР" равна количеству благоприятных исходов, деленному на количество возможных комбинаций :point_down:
```
P(ТОР) = 1 / 20 = 0.05 = 5%
```

## #3
#### Найти вероятность угадать 4 номера в спортлото 6 из 45
Вероятность угадать 4 номера в спортлото 6 из 45 будет равна :point_down:
```
P = C(45, 4) / C(45, 6)
```
Вычисляя это, получаем :point_down:
```
P = (45! / (4! * (45-4)!) ) / (45! / (6! * (45-6)!) )
P = (45 * 44 * 43 * 42) / (4 * 3 * 2 * 1 * 39 * 38)
P = 0.000181 = 0.0181%
```

## #7
#### В партии из 10 деталей 7 стандартных. Найти вероятность того, что среди 6 взятых наугад деталей 4 стандартных.
Общее число способов выбрать 6 деталей из 10 :point_down:
```
C(10, 6) = 10! / (6! * (10-6)!) = 10! / (6! * 4!) =
  = (10 * 9 * 8 * 7 * 6 * 5) / (6 * 5 * 4 * 3 * 2 * 1) = 210.
```
Число способов выбрать 4 стандартные детали из 7 :point_down:
```
C(7, 4) = 7! / (4! * (7-4)!) = 7! / (4! * 3!) =
  = (7 * 6 * 5 * 4) / (4 * 3 * 2 * 1) = 35.
```
Число способов выбрать 2 нестандартные детали из 3 :point_down:
```
C(3, 2) = 3! / (2! * (3-2)!) = 3! / (2! * 1!) = 3.
```
Вероятность того, что среди 6 взятых наугад деталей 4 стандартных.
```
P = (35 * 3) / 210 = 105 / 210 = 0.5 = 50%
```

## #8
#### Среди 25 студентов группы, в которой 10 девушек, разыгрывается 5 билетов. Найти вероятность того, что среди обладателей билетов окажутся 2 девушки.
Всего возможностей выбрать 5 студентов из 25 :point_down:
```
C(25, 5) = 25! / (5! * (25-5)!) = 53130
```
Количество способов выбрать 2 девушки из 10 и 3 студента из оставшихся 15 (мужчины) :point_down:
```
C(10, 2) * C(15, 3) = (10! / (2! * (10-2)!)) * (15! / (3! * (15-3)!)) = 45 * 455 = 20475
```
вероятность того, что среди обладателей билетов окажутся 2 девушки :point_down:
```
P = 20475 / 53130 ≈ 0.3858 = 38.6%
```
## #9
#### В ящике 2 белых, 3 красных и 5 голубых шаров. Наудачу извлекают 3 шара. Найти вероятность того, что все шары разного цвета.
Общее количество комбинаций :point_down:
```
C(10, 3) = 10! / (3! * (10-3)!) = 120
```
Выбрать 1 шар из 2 белых :point_down:
```
C(2,1) = 2! / (1! * (2-1)!) = 2 / 1 = 2
```
Выбрать 1 шар из 3 красных :point_down:
```
C(3,1) = 3! / (1! * (3-1)!) = 6 / 2 = 3
```
Выбрать 1 шар из 5 голубых :point_down:
```
C(5,1) = 5! / (1! * (5-1)!) = (4! * 5) / 4! = 5
```
Комбинации, в которых все 3 шара разного цвета :point_down:
```
C(2,1) * C(3,1) * C(5,1) = 2 * 3 * 5 = 30
```
Вероятность, что все 3 шара будут разного цвета :point_down:
```
P = 30 / 120 = 1/4 = 0.25 = 25%
```

Оценка    2  3 4 5
Студентов 7 11 7 5
